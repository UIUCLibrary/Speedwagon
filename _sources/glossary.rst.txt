========
Glossary
========

.. glossary::

    CMake
        A meta-build system primary used in many cross-platform C and C++
        applications.

        `CMake Homepage <https://cmake.org/>`_

    Exiv2
        Open-source software and code library for managing image metadata.

        `Exiv2 Homepage <https://www.exiv2.org/>`_

    Google Tesseract
        Open-Source software and code library from Google that performs :term:`OCR <Optical Character Recognition>`.

        `Google Tesseract Homepage <https://opensource.google.com/projects/tesseract>`_

    JPEG 2000
        Compression standard and file format for storing image data.

    Kakadu Encoder
        Propriety software for encoding :term:`JPEG 2000` files.

        `Kakadu Homepage <https://kakadusoftware.com/>`_

    Optical Character Recognition
        A processes that converts images of characters to text.

        `More information <https://en.wikipedia.org/wiki/Optical_character_recognition>`_

    Python
        Programming language used to develop Speedwagon

        `Python Homepage <https://www.python.org/>`_

    Python Package
        A collection of :term:`Python` code that is bundled up for end-user
        distribution or reuse in another Python project.

    Tessdata Directory
        The location on your computer for where Speedwagon can direct Tesseract
        to the language data files id needs to identify text characters within
        image files.

        This exact location on your computer can be found in the system
        Settings for Speedwagon.
