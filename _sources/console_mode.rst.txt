Show Console on Windows
+++++++++++++++++++++++

By default, when Speedwagon is run from the start menu, the console is hidden.
However, if you want to turn this on to get additional feedback, such as for
debugging purposes you can still do so.

#. Locate and open the properties of Speedwagon start menu shortcut.

    .. image:: shortcut_properties.png

#. Edit the Target property to include "/with_console" at the end

    .. image:: with_console.png
