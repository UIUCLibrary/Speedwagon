Command Line Arguments¶

usage: speedwagon [-h] [--version] [--starting-tab START_TAB] [--debug]
                  {run} ...

Positional Arguments¶

command

Possible choices: run

sub-command help

Named Arguments¶

--version

show program’s version number and exit

--starting-tab

Which tab to have open on start

--debug

Run with debug mode

Default: False

Sub-commands¶

run¶

run help

speedwagon run [-h] [--json JSON]

Named Arguments¶

--json

Run job from json file

speedwagon

Collection of tools and workflows for DS
